# InAmigos Foundation Website

A modern, responsive website for InAmigos Foundation - an Indian NGO based in Bilaspur, Chhattisgarh.

## About InAmigos Foundation

InAmigos Foundation was founded on September 23, 2020, with a mission to unite minds for change. Based in Bilaspur, Chhattisgarh, we work on projects focused on food security, environmental conservation, education, women empowerment, and animal welfare.

## Website Structure

- HTML5 semantic markup for accessibility
- CSS3 with modern styling
- Responsive design for all device sizes
- Clean, minimal aesthetic with print-inspired design

## Features

- Header with navigation
- Impact statistics and hero section
- Project showcase
- Organization credentials
- Call-to-action for donations and volunteering
